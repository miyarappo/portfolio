Project Details

不動産事業を運営する株式会社インデックスのWebアプリケーションのプロトタイピングとデザインを担当しました。

「マンション購入を検討するお客様とその場で収支シミュレーションをしたい」という要望を受け、必要な項目を精査しながらプロトタイプを作り込みました。

タブレットでストレスなく使えるデザインを目指しました。具体的には、入力項目と試算結果を分かりやすく表示し、画面遷移せず結果を確認出来るよう工夫しました。

  • 担当：プロトタイピング / デザイン
  • 使用ソフト・言語：Photoshop / HTML5 / CSS3 / JavaScript
  • 製作期間：2017年10月 （20h程度）
