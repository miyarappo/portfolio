Project Details

水回りの総合商社、株式会社ユアーズのコーポレートサイトのレスポンシブ対応を行いました。

Googleのモバイルファーストインデックスを導入に備えて、PC版サイトとモバイル版サイトを統一しました。

34ページを1週間という短い期間でコーディングし直しました。納期がに短い中で、前任者のCSS設計を考慮しながら品質と納期のバランスをとって完成させました。

  • 担当：コーディング
  • 使用ソフト・言語：HTML5 / CSS3 / JavaScript
  • 製作期間：2017年11月 （30h程度）
