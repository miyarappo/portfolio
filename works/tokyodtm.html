Project Details

音楽専門学校である東京DTM作曲音楽学校の新講座募集ページの ディレクション、デザイン、コーディングを担当しました。

ターゲットはDTM中級者~上級者を想定し、講座理解の促進と申込数の増加を目的としてしています。

ターゲットに近しいユーザーにヒアリングし、ユーザーの意思決定に必要な情報量を洗い出しました。 どうしても文字量を多くする必要があったため、ユーザーが見たい情報を負荷なく見ることが出来るよう工夫しました。 具体的には、読むことを阻害しないミニマルなデザインとし、ユーザーの必要性におじて情報の表示・非表示が出来るようにしました。

  • 担当：ディレクション / デザイン / コーディング
  • 使用ソフト・言語：Photoshop / Atom / HTML5 / CSS3 / JQuery
  • 製作期間：2017年8月~2017年9月 （150h程度）

情報量が多いので詳細をアコーディオンに。

ユーザーの目的に応じてフォームを出し分け。
